    Base.:∩ — Method.
    ∩(S1::Separator, S2::Separator)

    Separator for the intersection of two sets given by the separators S1 and S2. Takes an iterator of intervals (IntervalBox, tuple, array, etc.), of length equal to the total number of variables in S1 and S2; it returns inner and outer tuples of the same length

    IntervalConstraintProgramming.flatten! — Method.

    flatten! recursively converts a Julia expression into a "flat" (one-dimensional) structure, stored in a FlatAST object. This is close to SSA (single-assignment form, https://en.wikipedia.org/wiki/Static_single_assignment_form).

    Variables that are found are considered input_variables. Generated variables introduced at intermediate nodes are stored in intermediate. Returns the variable at the top of the current piece of the tree.

    IntervalConstraintProgramming.mul_rev_new — Method.

    ∘_rev1(b, c, x) is the subset of x such that x ∘ b is defined and in c ∘_rev2(a, c, x) is the subset of x such that a ∘ x is defined and in c
